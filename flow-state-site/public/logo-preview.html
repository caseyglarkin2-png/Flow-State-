YardFlow Logo Variants

CURRENT

Network Hub

Central hub with 3 connected facility nodes. Emphasizes network coordination and interconnected facilities.

Flow Arrows

Directional movement showing Viscosity → Leak → YNS narrative. Emphasizes friction reduction and throughput optimization.

Network Nexus

Triangle of 3 connected nodes. Shows multiple facilities as part of a network. Emphasizes network effect and enterprise scale.

Live Signal

Waveform/pulse pattern. Represents real-time intelligence, predictive ETA, and AI-powered orchestration. Tech-forward vibe.

How to Switch Logos

  1. Open lib/branding.ts
  2. Go to line 13
  3. Change ACTIVE_VARIANT to one of:
    • 'network' - Network Hub (current)
    • 'flow' - Flow Arrows
    • 'nexus' - Network Nexus
    • 'signal' - Live Signal
  4. Save, rebuild (npm run build), and push
  5. New logo appears everywhere: header, OG images, social cards
